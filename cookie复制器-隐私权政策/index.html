Cookie复制器 隐私权政策

最后更新日期：2026年1月18日

概述

Cookie复制器（以下简称"本扩展程序"）是一款帮助用户管理和复制浏览器Cookie的Chrome扩展程序。我们非常重视您的隐私，本隐私权政策旨在说明本扩展程序如何处理您的数据。

数据收集

本扩展程序不收集任何个人数据

本扩展程序：

  • 不收集任何个人身份信息
  • 不收集您的浏览历史记录
  • 不收集您访问的网站信息
  • 不收集您的Cookie数据
  • 不存储任何数据到远程服务器
  • 不与任何第三方共享任何数据

本地数据访问

本扩展程序仅在以下情况下访问您的数据，且所有操作均在您的本地设备上进行：

  1. Cookie访问：当您主动点击扩展程序图标时，本扩展程序会读取当前网页的Cookie信息，以便您查看和复制。这些数据仅在您的浏览器本地处理，不会被传输到任何外部服务器。

  2. 剪贴板访问：当您选择复制Cookie时，本扩展程序会将Cookie信息写入您的系统剪贴板。此操作完全在本地完成。

  3. Cookie管理：当您选择添加或清除Cookie时，本扩展程序会修改当前网站的Cookie。此操作仅影响您本地浏览器中的数据。

权限说明

本扩展程序请求以下权限，仅用于实现其核心功能：

权限 用途
activeTab 获取当前标签页的URL和域名信息，以便读取对应的Cookie
cookies 读取、添加和删除Cookie，这是本扩展程序的核心功能
host_permissions 允许在任何网站上使用Cookie管理功能

数据存储

本扩展程序不使用任何形式的数据存储，包括但不限于：

  • 本地存储 (localStorage)
  • IndexedDB
  • 云存储
  • 远程数据库

所有操作都是即时的，不会保留任何历史记录。

数据传输

本扩展程序不会将任何数据传输到外部服务器。所有功能均在您的浏览器本地执行。

第三方服务

本扩展程序不使用任何第三方服务，包括但不限于：

  • 分析服务
  • 广告服务
  • 追踪服务
  • 云服务

儿童隐私

本扩展程序不针对13岁以下的儿童，也不会有意收集任何年龄段用户的个人信息。

安全性

由于本扩展程序不收集或存储任何数据，因此不存在数据泄露的风险。所有Cookie操作都在您的本地浏览器环境中安全执行。

隐私政策的变更

如果我们对本隐私政策进行任何重大变更，我们将通过更新本页面来通知您。我们建议您定期查看本隐私政策以了解任何变更。

开源声明

本扩展程序是开源项目，您可以在 GitHub 上查看完整的源代码，以验证我们的隐私承诺。

联系我们

如果您对本隐私政策有任何疑问或建议，请通过以下方式联系我们：

  • GitHub Issues: https://github.com/xiaoyu-ai/CookiePlugin/issues

同意

使用本扩展程序即表示您同意本隐私政策的条款。
